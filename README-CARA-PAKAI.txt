===============================================
  CARA MENJALANKAN LEMBATA OCEAN EXPLORER
===============================================

Ada 3 cara untuk menjalankan aplikasi:

-------------------------------------------
1. CARA PALING MUDAH (REKOMENDASI)
-------------------------------------------
   Double click file: start-dan-buka.bat
   
   File ini akan:
   - Start server otomatis
   - Buka browser otomatis
   - Siap pakai!

-------------------------------------------
2. MANUAL (Start Server Dulu)
-------------------------------------------
   a. Double click: start-server.bat
   b. Tunggu sampai muncul "Ready"
   c. Double click: open-browser.bat
   
-------------------------------------------
3. MANUAL PENUH
-------------------------------------------
   a. Buka CMD di folder ini
   b. Ketik: npm run dev
   c. Tunggu "Ready"
   d. Buka browser: http://localhost:3000

-------------------------------------------
TROUBLESHOOTING
-------------------------------------------

Q: Server tidak jalan?
A: Pastikan Node.js sudah terinstall
   Ketik di CMD: node --version

Q: Port 3000 sudah dipakai?
A: Stop aplikasi yang pakai port 3000
   Atau restart komputer

Q: CSS tidak muncul?
A: Tekan CTRL + SHIFT + R di browser
   (Hard refresh)

Q: Foto tidak muncul?
A: Pastikan internet tersambung
   (Foto dari Google Maps)

-------------------------------------------
STOP SERVER
-------------------------------------------
Tekan CTRL + C di window server
Atau tutup window CMD yang ada tulisan
"Lembata Ocean Server"

-------------------------------------------
LOKASI FILE
-------------------------------------------
Server: D:\AI\ocean\lembata-ocean
Port: http://localhost:3000
Atau: http://192.168.0.101:3000
